## üöÄ About
This project is a personal portfolio website designed with a focus on **unique UI/UX** and **clean code**. Unlike standard bootstrap templates, this project uses custom CSS to create a distinct visual identity using an "Electric Teal" and "Deep Navy" color palette.

## ‚ú® Features
* **Modern Dark Mode:** A professional deep navy background (`#0a192f`) with high-contrast text.
* **Responsive Design:** Fully fluid layout using `clamp()` typography and Media Queries.
* **CSS Grid & Flexbox:** utilized for the "Offset Skills Grid" and layout structure.
* **Interactive Elements:** Custom hover states, reveal effects, and a list-view project section.
* **Zero Dependencies:** Built entirely with vanilla HTML and CSS (no frameworks).
